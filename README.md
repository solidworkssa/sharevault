# ShareVault

Encrypted file sharing registry.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/ShareVault.sol`
- Stacks: `contracts/stacks/contracts/sharevault.clar`

## Setup
```bash
pnpm install
pnpm dev
```
